# Logitech G305 — Battery Checker

Check your mouse battery percentage without installing G HUB.

Works on any Windows PC. No installs required.

## How to run

1. Open **PowerShell** (search "PowerShell" in the Start menu)
2. Copy the command below using the copy button, paste it in, and press **Enter**


The terminal will clear and show just your battery percentage.
