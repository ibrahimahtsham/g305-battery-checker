# Logitech G305 — Battery Checker

Check your mouse battery percentage without installing G HUB.

Works on any Windows PC. No installs required.

## How to run

1. Download **mouse-battery.txt**
2. Open it with Notepad
3. Press **Ctrl+A** to select all, then **Ctrl+C** to copy
4. Open **PowerShell** (search "PowerShell" in the Start menu)
5. Right-click inside the PowerShell window to paste, then press **Enter**

You'll see something like:

```
Battery: 90%  (Discharging)
```
